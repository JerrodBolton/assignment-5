# Assignment 6: Creating a GUI in Python

## Overview
In this assignment, we will learn how to create a Graphical User Interface (GUI) application in Python. The goal is to design a stand-alone GUI app that provides a great user experience and aligns with a specific theme of your choice.

---

## Guidelines
- **Create a stand-alone GUI application in Python.**
- Ensure your app is user-friendly and provides a great experience for others.
- Choose a theme for your app, and ensure all code supports that theme.

---

## Requirements / Expectations
1. **Develop a GUI application in Python.**
2. Use at least one GUI library:
    - Tkinter (recommended) or PyGame (depending on your interests).
3. **Comment your source code extensively** to explain its functionality.
4. Focus on creating a **great user experience**.
5. The app should not just be a demo; it should have a clear purpose and functionality.

---

## Deliverables / Submission
- Submit the **source code (.py file)** for your application.
- Include **screenshots** or a **YouTube link** to a video showcasing your app in action.
- **Do not zip your submission.**

---

## Notes
- Be creative and have fun with your project!
- Remember, the theme you choose should guide the design and functionality of your app.
